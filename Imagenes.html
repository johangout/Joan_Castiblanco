Hasta el momento solo hemos realizado páginas web planas, es decir solamente texto. Pero una página web hoy en día no solamente debe tener texto también tener sonidos, imágenes, videos con el objetivo de llamar la atención del visitante o usuario final. Etiqueta "img" Nos sirve para insertar una imagen y va acompañada de los siguientes atributos src para indicar la ubicación de la imagen y alt aporta un texto alterno que representa a la imagen para personas con discapacidad visual.
